Образование

Основно

Основното образование го завршив во ООУ „Гоце Делчев“ во Тетово со одличен успех од 5.00.

Средно

Средното образование го завршив во СОУ „Кирил Пејчиновиќ“ во Тетово со одличен успех од 5.00.

Факултет

Високото образование го продолжив на Факултетот за информатички науки и комјутерско инженерство во Скопје и моментално сум прва година

Курсеви кои ги имам посетувано

Курсеви C++

Курсевите за програмскиот јазик C++ ги започнав во 2016 година. Ги посетував 6 години и имам освоено 8 сертификати.

Курсеви за англиски јазик

Курсевите за англиски јазик ги имам започнато од прво одделение се до четврта година средно. Англискиот јазик го познавам на напредно ниво.

Курсеви за Java Script

Курсевите за Java Script ги посетував во 2022 година. Со овие курсеви се стекнав со основно познавање на програмскиот јазик Java Script.

Курсеви за Python

Курсевите за Python ги посетував во 2022 година. Со овои курсеви се стекнав со основно познавање на програмскиот јазик Python и негови библиотеки како pandas.

Курс за дигитална писменост

Курсот го имам посетувано две години во 2019 и 2020 година. Со овој курс стекнав заење за многу алатки на интернет, користење на weebly, едитирање на слики и унапредување на моите способности за изразување.
